Features:
Supports basic operations: +, -, *, /, ** (power), % (modulo)
Handles invalid inputs (non-numbers, invalid operators)
Prevents division by zero errors
Continues calculations until user chooses to exit
Uses floating-point division for accurate results
How to Use:
Save code in calculator.rb
Run with: ruby calculator.rb
Follow prompts to enter numbers and operations
Choose whether to continue after each calculation
This implementation includes:
A Calculator class with methods for each operation
Input validation functions
Error handling for division by zero
Loop control for repeated calculations
Clear user prompts and formatted output
The calculator will:
Accept integers and decimals
Show clear error messages for invalid inputs
Handle edge cases like division by zero
Maintain state between calculations
Support exponentiation and modulo operations
